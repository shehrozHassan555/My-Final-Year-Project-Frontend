
# Ocular Disease Detection – Frontend

# Overview

This repository contains the frontend of the **Ocular Disease Classification System**, developed as part of a Final Year Project at Lahore Garrison University.

The frontend provides a clean and user-friendly interface that allows users to upload retinal fundus images, view prediction results, and visualize Explainable AI (XAI) heatmaps generated by the backend model.

It is designed to ensure simplicity, accessibility, and a smooth user experience.

# Purpose of Frontend

The frontend is responsible for:

* Allowing users to upload retinal images
* Sending images to the backend API
* Displaying disease prediction results
* Showing confidence scores
* Displaying Grad-CAM heatmap visualizations
* Providing an intuitive medical-style UI

 Features

 Image Upload Interface
 Prediction Display Section
 XAI Heatmap Visualization
 Responsive Design
 Clean and professional layout

# Technologies Used

* HTML5
* CSS3
* Bootstrap (if used)
* JavaScript
* Flask Templates (Jinja2)

# How It Works

1. User uploads a retinal image.
2. Frontend sends the image to the Flask backend.
3. Backend processes the image using EfficientNet-B0 model.
4. Prediction result and Grad-CAM heatmap are returned.
5. Frontend displays:

   * Predicted disease class
   * Confidence percentage
   * Visual explanation heatmap
# UI Design Goals

* Minimal and professional medical theme
* Easy navigation
* Clear result presentation
* Fast image preview
